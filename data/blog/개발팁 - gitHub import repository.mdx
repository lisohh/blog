# gitHub repository organization에 import할 때

이번 프로젝트에서 프론트엔드와 백엔드가 따로 레포지토리를 만들게 되면서 <br/>
한꺼번에 보여줄 수 있는 방법이 무엇이 있을까 하다가 <br/>
organization을 만들어서 레포지토리를 한 곳에 모아놓으면 좋겠다는 조언을 얻었다.<br/>

그래서 organization을 만들고 레포지토리를 import하려는데, 은근히 해매서<br/>
organization 계정에 레포를 만들어야 하는데 내 개인 계정에 다시 만드는 실수를 했다.<br/>

혹시 나같은 사람이 있을까봐 적어보는 레포지토리 Import tip! <br/>

1. import할 레포지토리에 들어가 clone하는 url주소를 복사한다.
   ![import-guide-1](/static/images/import1.png)
   초록색 `code`버튼을 누르고 `HTTPS` 탭에 보이는 주소 오른쪽에 두개의 사각형이 겹쳐진 버튼을 누르면 주소가 복사된다.<br/>
2. 레포지토리를 `import`할 `organization`으로 이동해서 우측의 import 버튼을 누른다.
   ![import-guide-2](/static/images/import2.png)
3. 아까 복사해온 주소를 `Your old repository's clone URL` 칸에 붙여넣는다.
   ![import-guide-3](/static/images/import3.png)
4. 중요!! 이제 가져온 레포지토리가 `import`될 위치와 `import`되었을 때의 새 이름을 지어주어야 한다.
   여기서 나는 내가 `Owner`니까 내 계정으로 `import`해야지 했는데, <br/>
   그게 아니라 레포지토리가 `import`될 위치를 골라주어야 한다. <br/>
   그래서 화살표 버튼을 누르면 아래에 쭉 내가 속한 `organization`들이 뜨는데 그중에 고르면 된다.<br/>
   그리고 `import`된 위치에서 보이는 새 이름을 지어주면 된다. => 기존 레포지토리 이름이랑 달라야함 <br/>
   다 작성했으면 `Begin import`버튼을 눌러준다.
5. 그럼 아래와 같은 이미지가 보이고, 무지개빛 로딩이 완료되면 새로 `import`된 레포지토리 주소가 쫜! 하고 뜬다.
   ![import-guide-4](/static/images/import4.png)
