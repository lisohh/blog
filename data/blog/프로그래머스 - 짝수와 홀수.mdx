## 🍑 링크

[짝수와 홀수](https://school.programmers.co.kr/learn/courses/30/lessons/12937)

## 🍋 문제 설명

> 정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.

## 🍐 제한 조건

> num은 int 범위의 정수입니다.
> 0은 짝수입니다.

## 🍊 풀이

```tsx
function solution(num) {
  const number = Math.abs(num % 2)
  return number == 1 ? 'Odd' : 'Even'
}
```

범위가 int 범위의 정수이므로, 나눈 값이 음수(-)가 될 수도 있으니
[`Math.abs`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/abs)를 사용해서 나눗셈의 나머지값에 절대값을 구한다.

절대값을 변수 number에 넣고

삼항연산자로 만약 number가 1이면
홀수이므로 "Odd"를 리턴하고
아니라면 짝수이므로 "Even"을 리턴한다.
