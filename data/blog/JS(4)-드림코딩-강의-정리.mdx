## 연산.반복문(4강)

다른 언어에서도 공통적으로 쓸 수 있는 operators~

### 1. String concatenation (문자 연결)

```css
console.log('my' + ' cat');
console.log('1' + 2);
console.log(`string literals: 1 + 2 = ${1 + 2}`);
```

### 2. Numeric operators (숫자 연산자)

```css
console.log(1 + 1); // add
console.log(1 - 1); // substract
console.log(1 / 1); // divide
console.log(1 * 1); // multiply
console.log(5 % 2); // remainder
console.log(2 ** 3); // exponentiation
```

### 3. `++`, `--` operators

```css
let counter = 2;
const preIncrement = ++counter;
// counter = counter + 1;
// preIncrement = counter;
console.log(`preIncrement: ${preIncrement}, counter: ${counter}`);
const postIncrement = counter++;
// postIncrement = counter;
// counter = counter + 1;
console.log(`postIncrement: ${postIncrement}, counter: ${counter}`);
const preDecrement = --counter;
console.log(`preDecrement: ${preDecrement}, counter: ${counter}`);
const postDecrement = counter--;
console.log(`postDecrement: ${postDecrement}, counter: ${counter}`);
```

### 4. `=` operators

```css
let x = 3;
let y = 6;
x += y; // x = x + y;
x -= y;
x *= y;
x /= y;
```

### 5. `<=` operators

```css
console.log(10 < 6); // less than
console.log(10 <= 6); // less than or equal
console.log(10 > 6); // greater than
console.log(10 >= 6); // greater than or equal
```

### 6. Logical operators (`||`, `&&`, `!`)(논리 연산자)

<mark>비교 연산자는 true값을 찾으면 그 뒤로는 연산을 하지 않는다.</mark>
<br />
그러므로 간단한 value값을 제일 먼저 놓는게 효율적이다. 무거운 expression은 맨 뒤에 놓기~!

### 7. Equality Operators (`==`, `===`)

== loose equality, with type conversion
타입까지는 비교하지 않는다.

=== strict equality, no type conversion
타입까지 비교한다!

object equality by reference

```css
const ellie1 = { name: 'ellie' };
const ellie2 = { name: 'ellie' }; //ellie1과 ellie2는 서로 다른 레퍼런스를 가리킨다.
const ellie3 = ellie1;
console.log(ellie1 == ellie2); //reference값이 다르므로 false
console.log(ellie1 === ellie2); //reference값이 다르므로 false
console.log(ellie1 === ellie3); //reference값을 ellie1에 할당한 것이라 true
```

```css
// equality - puzzler
console.log(0 == false); //값이 다르므로 true
console.log(0 === false); //타입이 다르므로 false
console.log('' == false); //true
console.log('' === false); //타입이 다르므로 false
console.log(null == undefined); //true
console.log(null === undefined); //타입이 다르므로 false
```

### 9. Ternary operators (삼항 연산자)

### 13. for loop (for 반복문)
```css
//for loop, for(begin; condition; step)
for (i = 3; i > 0; i--) {
  console.log(`for: ${i}`);
}

for (let i = 3; i > 0; i = i - 2) {
  // inline variable declaration
  console.log(`inline variable for: ${i}`);
}
````

### 14. nested loop (중첩 반복문)

실행 속도가 느려서 되도록 쓰지 않는게 좋다.

```css
for (let i = 0; i < 10; i++) {
  for (let j = 0; j < 10; j++) {
    console.log(`i: ${i}, j:${j}`);
  }
}
```
