map()반복문을 사용해서 단어 수만큼의 li를 만들어 주면 좋다.<br/>
map()은 배열을 받아서 또 다른 배열을 반환하는 건데 이때 반환되는 배열의 요소는 jsx로 작성하면 된다.<br/>
화살표 함수로 바로 return문을 작성한다.

```js
import React from 'react'
import dummy from '../db/data.json'

export default function DayList() {
  return (
    <ul className="list_day">
      {dummy.days.map((day) => (
        <li key={day.id}>Day {day.day}</li>
        //key는 반복되는 요소에 고유한 값을 넣어줘야 하는데
        //여기서는 day.id값을 넣어준다.
      ))}
    </ul>
  )
}
```

마찬가지로 dummy 데이터를 사용하는 Day.js파일도 component폴더안에 만들어준다.<br/>
그리고 App.js에 Day.js를 import해준다.

```js
import React from 'react'
import dummy from '../db/data.json'

export default function Day() {

    <div>

              <td>{word.eng}</td>
              <td>{word.kor}</td>
            </tr>
          ))}
        </tbody>
      </table>
    </div>
  )
}
```
