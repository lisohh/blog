# 유용한 깃 커맨드

- `git reset --soft HEAD^`
  - 최근 커밋 한 개(`HEAD^`)를 취소하되, 변경 내용은 보존(`--soft`)
- `git commit --amend --no-edit`
  - 최근 커밋 한 개에 현재 staging된 변경사항을 병합
  - 메세지는 수정하지 않음(`--no-edit`)
