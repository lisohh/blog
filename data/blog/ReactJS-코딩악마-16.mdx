# 마무리 및 부록 : 타입스크립트 적용하기

느린 인터넷 환경을 써보려면 개발자 도구 - 네트워크 탭에서 slow 3G를 선택한다.<br/>
단어가 로딩중일 때는 로딩중이라는 메시지를 보여준다.<br/>
통신 중일 때는 버튼을 누르지 못하게 만들어준다. => 오류 방지

## 타입스크립트 적용

1. 타입스크립트 설치
   `npm install typescript @types/node @types/react @types/react-dom @types/jest @types/rect-router-dom`입력

2. js는 tsx로 변경

3. interface를 만들고 type 설정
   절대 any 타입을 남용하면 안된다.<br/>
   type명 뒤에 '?'를 붙이면 그 type은 optional하다는 의미이다.
   interface 앞에 export를 붙여주면 그 interface는 다른 컴포넌트에서도 사용 가능하다.
